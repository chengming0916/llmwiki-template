# LINT - 检查知识库完整性

检查 `wiki/` 目录下的页面完整性，发现潜在问题。

## 输入

`/lint [范围]`

- `/lint` - 检查整个 wiki/ 目录
- `/lint concepts` - 只检查概念页面
- `/lint entities` - 只检查实体页面
- `/lint sources` - 只检查源摘要页面

## 检查项目

1. **孤立页面**：查找没有任何页面链接到它的页面
2. **断链**：查找指向不存在页面的链接
3. **信息矛盾**：检测页面间的信息不一致
4. **重复内容**：查找相似或重复的页面
5. **缺失链接**：检查是否缺少应有的双向链接

## 规则

- **禁止修改**：只报告问题，不自动修复
- **详细定位**：报告问题所在的文件和位置
- **修复建议**：为每个问题提供具体修复建议

## 输出格式

```
## LINT 报告

### 发现问题

| 类型 | 文件 | 问题描述 | 修复建议 |
|------|------|----------|----------|
| 孤立页面 | wiki/concepts/xxx.md | 没有任何页面链接到此页面 | 添加与其他页面的链接 |

### 统计

- 总页面数：N
- 孤立页面：N
- 断链：N
- 待处理建议：N
```
