# llmwiki-template

#### Description
基于 Andrej Karpathy LLM WIKI 理念搭建的个人知识库模板，遵循Raw/Wiki/Schema三层架构，摒弃传统 RAG 临时检索模式，以大模型编译沉淀结构化 Markdown 维基，支持 AI 自主维护、知识互联与持续迭代，开箱即用可快速搭建专属 LLM 知识库。
