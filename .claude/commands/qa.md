# QA - 知识库问答

基于 `wiki/` 目录中的内容回答问题。

## 输入

`/qa <问题>`

示例：`/qa 什么是双向链接`

## 工作流程

1. **搜索知识库**：在 `wiki/` 目录中查找相关信息
2. **分析关联**：查看相关页面的链接关系
3. **综合回答**：基于找到的内容整理答案
4. **标注来源**：引用具体的页面和位置

## 规则

- **禁止幻觉**：只回答知识库中存在的内容
- **标注来源**：为每个观点标注引用页面
- **信息不完整时**：说明情况并建议使用 `/ingest` 添加内容
- **诚实回答**：不确定时明确说明，不猜测

## 输出格式

```
## QA 结果

### 问题
xxx

### 答案
xxx

### 来源
- [[wiki/xxx]] - 相关内容
- [[wiki/yyy]] - 补充信息

### 建议
如需更多信息，建议：/ingest raw/xxx.md
```
